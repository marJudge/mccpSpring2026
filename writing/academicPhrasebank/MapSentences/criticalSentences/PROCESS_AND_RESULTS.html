Pipeline (4 steps)

1Extract sentences from HTML

Script reads all .html files in html_collection, parses LaTeXML <p class="ltx_p"> paragraphs, and keeps lines ≥ 15 characters.

Command: python3 find_critical_sentences.py --extract-only
Output: sentences_all.json — 28 papers, 6,601 sentences
2Filter candidates (keywords)

Each sentence is checked for phrases like previous stud, prior work, little attention, few studies, lack of, failed to address, limited to, contrasts with, overlooked, etc.

Command: python3 find_critical_sentences.py --candidates
Output: candidates.tsv — 128 candidate sentences (paper_id, sentence)
3Classify & extract templates (AI, no API)

In-session AI read candidates.tsv, assigned each sentence to Category 1 (inadequacies of previous studies) or Category 2 (paucity/lack of previous research), dropped false positives, and wrote a short extracted template (X, Y, Z placeholders) per match.

Output: matches_from_ai.json
69 matches: 50 inadequacies, 19 paucity/lack
4Generate reports

Script loads matches_from_ai.json and findCriticalSentences.md, then writes the sentence report and a dark-theme HTML table view.

Command: python3 find_critical_sentences.py --from-matches matches_from_ai.json
Output: report_criticalSentences.md, report_criticalSentences.html
